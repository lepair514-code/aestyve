사용 방법

✅ 저장하면 홈페이지에 즉시 반영됩니다.
   (같은 브라우저에서 접속한 홈페이지에 바로 적용됩니다)

1. 사이드바에서 수정할 섹션을 선택하세요.
2. 내용을 편집 후 저장 버튼을 클릭하세요.
3. 저장 후 상단 홈페이지 버튼을 눌러 결과를 확인하세요.

⚠️ 다른 기기/브라우저에도 반영하려면?
   Import/Export 메뉴 → JSON 다운로드 → GitHub의
   data/content.json 파일을 교체하면 전체 반영됩니다.
