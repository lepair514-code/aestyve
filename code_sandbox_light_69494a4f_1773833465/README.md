# Aestyve 브랜드 홈페이지

> 메디큐브 스타일의 더마 커머스형 정적 홈페이지 + 간편 관리자(Admin)  
> **4개 언어 지원**: 🇰🇷 한국어 | 🇺🇸 영어 | 🇨🇳 중국어(간체, zh-CN) | 🇹🇭 태국어

---

## 📁 파일 구조

```
index.html          ← 메인 홈페이지
admin.html          ← 관리자(CMS) 페이지
css/style.css       ← 전체 스타일
js/main.js          ← 홈 렌더링 + i18n (zh-CN 포함)
js/admin.js         ← 관리자 CRUD
images/logo.png     ← 에스티브 로고 (업로드된 파일)
README.md           ← 이 문서
```

## 🌐 다국어(i18n) 적용 방식

### 언어 코드
| 버튼 | 언어 | 코드 | DB 필드 접미사 |
|-----|-----|------|--------------|
| 🇰🇷 | 한국어 | `ko` | `_ko` |
| 🇺🇸 | 영어 | `en` | `_en` |
| 🇨🇳 | 중국어(간체) | `zh-CN` | `_zh_CN` |
| 🇹🇭 | 태국어 | `th` | `_th` |

> **중요**: 중국어는 **간체(zh-CN)** 기준으로 통일. DB 필드명은 `title_zh_CN`, `content_zh_CN` 등으로 저장됩니다.

### 언어 결정 순서 (우선순위)
1. `?lang=ko|en|zh-CN|th` URL 파라미터 (최우선)
2. `localStorage('aestyve_lang')` — 사용자가 선택한 언어 저장
3. 브라우저 언어(`navigator.language`) 자동 감지
   - `zh`, `zh-CN`, `zh-TW`, `zh-HK` → `zh-CN` (간체 우선)
   - `th` → `th`
   - `ko` → `ko`
   - 나머지 → `en`

### Fallback 규칙
특정 언어 텍스트가 비어있으면 `ko → en` 순으로 폴백 처리됩니다.

## ⚙️ 관리자(admin.html) 사용법

### 접근 방법
- 브라우저에서 `admin.html` 직접 접속
- 좌측 사이드바에서 원하는 섹션 클릭

### 언어 탭 구조
모든 텍스트 입력에 4개 언어 탭이 제공됩니다:
```
[🇰🇷 한국어] [🇺🇸 EN] [🇨🇳 中文(简)] [🇹🇭 ไทย]
```

### 관리 항목
| 메뉴 | 기능 |
|------|------|
| **사이트 설정** | 브랜드명, 주색상, 공지바, 슬로건, 브랜드스토리, 통계, 연락처, 주소, SNS URL |
| **히어로 배너** | 이미지 URL, 제목/부제목/버튼(4개 언어), 배경색, 정렬, 순서, 노출 |
| **카테고리** | 이미지, 이름/피부고민/라벨(4개 언어), 링크, 순서, 노출 |
| **제품** | 이미지, 제품명/설명/배지(4개 언어), 가격, 탭 카테고리, 순서, 노출 |
| **리뷰** | 작성자, 평점, 내용/제품명(4개 언어), 노출 |

### 저장 반영 방법
1. admin.html에서 수정 후 **저장** 버튼 클릭
2. `index.html` 새로고침 → **즉시 반영**됩니다

## 🚀 배포

**Publish 탭**에서 클릭 한 번으로 라이브 URL이 발급됩니다.

배포 후 URL 파라미터로 언어를 강제 지정할 수 있습니다:
- `https://your-site.com/?lang=zh-CN` → 중국어(간체)로 오픈
- `https://your-site.com/?lang=th` → 태국어로 오픈

## 🗄️ 데이터 구조 (API 테이블)

### site_settings
브랜드 설정값. 레코드 1개만 사용(`id: site_001`).
- 색상, 공지바, 슬로건, 브랜드스토리, 통계, 푸터 정보, SNS URL

### banners
히어로 슬라이더 배너. `sort_order`로 순서 관리.
- 다국어 필드: `title_ko|en|zh_CN|th`, `subtitle_*`, `btn_text_*`

### categories
피부고민별 라인 카드. `sort_order`로 순서 관리.
- 다국어 필드: `name_*`, `skin_concern_*`, `label_*`

### products
제품 목록. `tab_category`로 탭 필터(`all|device|skincare|set`).
- 다국어 필드: `name_*`, `desc_*`, `badge_*`

### reviews
고객 리뷰. 무한 스크롤 트랙에 표시.
- 다국어 필드: `content_*`, `product_*`
